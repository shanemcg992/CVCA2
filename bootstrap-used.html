    • Bootstrap Layout:
    • container-fluid: For full-width containers.

    Bootstrap Utilities:

    • mb-3, mb-4, mb-5: For bottom margins.
    • mt-2, mt-5: For top margins.

    Bootstrap Javascript Components:

    • data-toggle="modal": For toggling modals.
    • data-target="#someModalID": For targeting modals to open.
    • data-dismiss="modal": For dismissing modals.
    • data-toggle="collapse": For toggling collapsible elements.
    • data-target="#someCollapseID": For targeting collapsible elements to open or close.
    • aria-expanded="true", aria-controls="someCollapseID": Accessibility attributes related to collapsible elements.
    • data-parent="#parentID": For accordion-style collapsible elements.
